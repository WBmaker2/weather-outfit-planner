# 오늘의 날씨 코디네이터: 사계절 옷장 (Weather Outfit Planner)

초등 1~2학년 통합교과 활동용으로 만든 날씨 기반 옷차림 게임입니다.  
미션(비/바람, 맑음 더움, 눈/추위, 봄바람 먼지)에 맞는 옷과 준비물을 골라 캐릭터에 장착한 뒤, **외출하기** 버튼으로 결과를 스스로 점검합니다.

## 수업 연결

- 대상: 1~2학년군 **바른 생활 / 슬기로운 생활**
- 성취기준: **[2슬04-01]** 날씨와 계절의 변화를 관찰하고 생활과 연결해 탐구한다.  
- 성취기준: **[2바04-01]** 날씨와 계절에 맞는 생활 수칙을 알고 실천한다.
- 활동 의도: 날씨 관찰 → 필요한 옷차림 판단 → 실천 연결까지 이어지도록,  
  실생활 준비 습관(보호, 체온 조절, 음수, 위생)에 대한 즉시 피드백을 제공합니다.

## 실행

```bash
npm install
npm run dev
```

## 릴리스 노트 (0.1.0)

- 비/바람, 맑은날 더움, 눈/추위, 봄바람 먼지 미션을 제공합니다.  
- 옷과 준비물을 클릭 또는 드래그해서 캐릭터에게 입힐 수 있습니다.  
- **외출하기**로 점검하면 “완벽한 외출 준비 끝!” 또는 미비한 항목/부적절 항목 안내를 제공합니다.  
- 생활 수칙 보기, 수업 모드, 다시 도전 흐름으로 교실 진행을 보조합니다.
- 데스크톱·태블릿·휴대폰 폭에서 핵심 조작(미션 확인, 아이템 선택, 외출하기)이 보이도록 반응형 레이아웃을 적용했습니다.  
- 로그인, 서버 저장소, 인증/로그인 절차는 없습니다. 수업에서 즉시 사용 가능한 브라우저 앱입니다.
