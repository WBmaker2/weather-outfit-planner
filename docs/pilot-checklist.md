# 파일럿 수업 점검표

## 수업 전

- [ ] GitHub Pages 공개 URL이 열리는지 확인한다.
- [ ] 교실 화면에서 수업 모드 켜기 버튼이 잘 보이는지 확인한다.
- [ ] 수업 모드에서 1~4단계 수업 흐름과 진행률이 잘 보이는지 확인한다.
- [ ] 교사용 오늘의 날씨 만들기에서 비, 바람, 더움, 먼지 조건을 조합해 미션이 만들어지는지 확인한다.
- [ ] 마우스 클릭 또는 터치로 아이템 선택이 되는지 확인한다.
- [ ] 생활 수칙 보기를 열어 문장이 수업 화면에서 읽히는지 확인한다.

## 수업 중

- [ ] 학생이 날씨 미션을 보고 필요한 준비물을 말할 수 있다.
- [ ] 교사가 만든 복합 날씨 미션에서도 체크리스트가 알맞게 바뀐다.
- [ ] 학생이 아이템을 직접 선택하거나 친구에게 선택 이유를 설명할 수 있다.
- [ ] 학생이 발표 질문 카드를 보고 선택한 이유를 한 문장으로 말할 수 있다.
- [ ] 외출하기 결과를 보고 부족한 준비물을 다시 고를 수 있다.
- [ ] 다시 도전 버튼으로 같은 미션을 한 번 더 시도할 수 있다.

## 수업 후

- [ ] 학생들이 헷갈린 날씨와 준비물을 기록한다.
- [ ] 다음 수업에서 실제 날씨 사진이나 교실 창밖 관찰과 연결할지 결정한다.
- [ ] 추가하고 싶은 아이템이나 미션을 메모한다.
